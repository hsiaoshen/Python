jQuery 是一个高效、精简并且功能丰富的 JavaScript 工具库。它提供的 API 易于使用且兼容众多浏览器，这让诸如 HTML 文档遍历和操作、事件处理、动画和 Ajax操作更加简单。本中文文档适用jQuery 1.0 - jQuery 3.1，翻译至官方文档，并且跟随官方文档持续更新。如果你是一个 jQuery 新手，我们建议你先到jQuery 学习中心练习一下。

在jQuery 3.0中，API的方面有更改。主要处理更改了 deferred（延迟），data（数据），show（显示）/hide（隐藏），并且删除一些已弃用的API。提供了一个jQuery Migrate 插件来帮助开发人员具有过渡升级到最新版本。更新详细信息请查看jQuery Core 3.0 升级指南 和 发布说明/更新日志。如果你以前使用的是1.9 版本之前的 jQuery ，建议你再看一看 1.9 版本升级指南。如果你经常需要升级新版本的 jQuery ，务必阅读jQuery官方博客上列出的版本发布说明。
